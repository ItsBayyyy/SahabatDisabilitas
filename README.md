## ğŸ® Cara Menggunakan Fitur Interaktif

### **1. Mode Kontras Tinggi**
- Klik tombol "Kontras Tinggi" di navigation bar
- Website berubah ke skema hitam-kuning
- Klik lagi untuk kembali ke mode normal

### **2. Simulasi Screen Reader**
- Klik tombol "Dengarkan Simulasi Screen Reader" di hero section
- Atau gunakan player di section Simulasi
- Audio akan memainkan simulasi screen reader

### **3. Tantangan Keyboard**
- Klik "Mulai Tantangan" di section Keyboard Challenge
- Gunakan tombol Tab untuk navigasi
- Coba akses semua 6 kartu tanpa mouse
- Gunakan Enter/Space untuk interaksi
